# Données météorologique à l'aviation en format IWXXM

L'Organisation météorologique mondiale (OMM), sous le parrainage de l'[Organisation de l'Aviation Civile Internationale (OACI)](https://www.icao.int/fr), a créé le modèle d’échange d’informations météorologiques de l’OACI ([IWXXM](https://community.wmo.int/iwxxm)) afin de rapporter les informations météorologiques pour l'aviation en langage de balisage extensible (XML) et plus précisément en langage de balisage géographique (GML).

IWXXM comprend des représentations pour les produits normalisés de l'Annexe 3 de l’OACI - Service météorologique pour la navigation aérienne internationale et le document n° 49 de l’OMM - Vol II Service météorologique pour la navigation aérienne internationale. IWXXM est conçu pour les échanges opérationnels d’informations météorologiques de machine à machine. Contrairement aux formes traditionnelles de code alphanumérique (TAC) des produits de l'annexe 3 de l'OACI / OMM n° 49, IWXXM n'est pas destiné à être lu directement par un utilisateur; il est conçu pour être ingéré par un logiciel agissant au nom de l'utilisateur des données, tel qu'un logiciel d'affichage ou des services de fonctionnalités Web.

Chaque message IWXXM contient tous les composants qui décrivent complètement le produit, par exemple, l'heure d'émission, l'état de l'émission (routine, modification, etc.) et les paramètres météorologiques spécifiques, y compris les unités de mesure, qui constituent le produit spécifique de l'annexe 3 diffusé. Les éléments supplémentaires qui se retrouvent dans les bulletins météorologiques d’aviation canadiens et qui ne sont pas décrits dans l’Annexe 3, sont toutefois inclus dans le modèle IWXXM à l'aide de schémas d'extension.
